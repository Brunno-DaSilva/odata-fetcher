Cognigy OData Viewer

Fetch and explore session analytics data from your Cognigy OData endpoint

How to store LLM tokens: Place an Overwrite Analytics node just above the "End" node in your flow. Set custom1=inputTokens, custom2=outputTokens, custom3=totalTokens using {{input.aiAgentOutput.tokenUsage.*}}. The analyzer sums all token values per session.

Returns turn-by-turn analytics records. Each row = one conversation turn. Fields include inputText, outputText, intent, rating, custom1–5.
